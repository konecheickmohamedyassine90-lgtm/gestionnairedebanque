# Gestionnaire de Banque

## Description

Ce projet Java a été réalisé dans le cadre du chapitre sur l’héritage.

L’application permet de gérer différents types de comptes bancaires grâce à l’utilisation de l’héritage en Java.

## Fonctionnalités

* Création d’un compte bancaire parent
* Héritage entre plusieurs classes
* Gestion des :

  * Comptes courants
  * Comptes épargne
  * Certificats de dépôt
* Affichage des informations des comptes
